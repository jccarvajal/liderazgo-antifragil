# Guía de Contribución

Este libro no es un texto estático; es un **documento vivo**. La entropía organizacional nunca descansa, y las formas en que la burocracia se disfraza de gestión evolucionan constantemente. Este repositorio aspira a mantenerse como una referencia de ingeniería forense para el diseño de organizaciones en 2026 y más allá.

Si eres un Arquitecto Organizacional, Gerente de Operaciones, Auditor Interno o Ingeniero de Sistemas, tu experiencia es valiosa para endurecer este marco de antifragilidad.

## ¿Cómo puedes colaborar?

### 1. Reporte de Erratas y Desajustes Conceptuales
Si encuentras un error técnico, una inconsistencia lógica o una definición que no cumple con el rigor del **Glosario Maestro**:

* Abre un **Issue** en GitHub describiendo el hallazgo.
* Si es algo simple (ortografía o formato), puedes realizar un **Pull Request** directo.

### 2. Nuevos Patrones y Vicios
El catálogo de **Patrones de Falla (Anexo C)** y **Vicios de Implementación (Anexo F)** siempre puede crecer. ¿Has detectado una nueva forma en que las organizaciones simulan competencia o abdican responsabilidad?

* Abre una discusión en la pestaña **Discussions** del repositorio.
* Propón el nuevo patrón siguiendo la estructura forense: *Síntoma Visible, Diagnóstico Técnico, Causa Raíz y Corrección*.

### 3. Estudios de Caso (Anónimos)
La teoría se valida en la trinchera. Si tienes ejemplos reales (anonimizados) de colapsos por *SPOF* humanos o éxitos de *Juicio Distribuido*:

* Envía una propuesta para incluirlos como "Cajas de Estudio" dentro de los capítulos correspondientes.

## Estilo y Formato
Para mantener la coherencia del "Contrato Semántico" de este manual:

* **Tono:** Forense, quirúrgico y libre de jerga motivacional. Evita palabras vacías como "empoderar", "sinergia" o "pasión". Usa términos de ingeniería: "latencia", "fricción", "diseño".
* **Formato:** Markdown estándar. Usa cajas de alerta (`!!! tip`, `!!! warning`, `!!! failure`) para resaltar axiomas y advertencias de diseño.
* **Filosofía:** Prioriza siempre la **estructura** sobre la intención. No juzgamos la moral de las personas, sino los incentivos del sistema.
